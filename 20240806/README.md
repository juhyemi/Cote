  [은상]

  - 따라서 `(x + 2) * (y + 2) = 전체 카펫의 넓이(brown + yellow)`가 되는 `(x, y)` 순서쌍을 탐색
    - 이 때 `(x, y)` 순서쌍은 yellow의 약수
