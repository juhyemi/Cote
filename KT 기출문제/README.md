| 배열 | 결과 |
| --- | --- |
| abcdef? | 2 |
| bbab? | 3 |

| abcd? | 2 |
| cacsdc? | 4 |
| cacabc? | 5 |
| cacsz? | 3 |
| 출력값 | 25 |

| A 문자열 | 결과 | 문자수 |
| --- | --- | --- |
| a | 2 | 1 |
| bbab? | 3 | 5 |

| cacsd | 4 | 5 |
| caca | 5 | 4 |
| cacsz | 5 | 5 |
